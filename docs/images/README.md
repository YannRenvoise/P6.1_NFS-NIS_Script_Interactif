8. `08-client-su-demo.png` : connexion `su - demoNIS` cote client.
9. `09-delete-user-server.png` : suppression de `demoNIS` cote serveur.
10. `10-client-user-removed.png` : verification que `demoNIS` n'apparait plus cote client.
